# Momentum_JS

Vanilla JS version of the productivity app <Momentum> extension on Chrome (https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca)

## Features

- [x] Clock
- [x] To Do List
- [x] Weather
- [x] Offline Support
- [ ] Daily Quote
- [ ] Daily Photo
- [ ] Add Function to Load Image Faster
